#qt简易播放器
## 依赖
1.qt5.20
2.vc++库
##  基本功能
> * 实现基本播放，上一曲，下一曲，歌曲列表，歌词显示。
> * 歌词显示，歌曲获取默认D:/MP3。
> * 拓展功能有待开发。
> * QT5开发，自带播放器函数
